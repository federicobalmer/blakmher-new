# Blakmher Ecommerce Frontend

Premium editorial homepage for Blakmher, a minimalist black home decor brand.

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## Scope

This initial version includes the homepage only:

- Hero section
- Category blocks
- Featured products
- Editorial brand section
- Minimal footer

No backend, payments, cart, login, database, or admin panel are included yet.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
